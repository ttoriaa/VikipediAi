读图提示：谁和谁直接竞争？

  • 模型层：DeepSeek、GLM 与 OpenAI、Gemini、Anthropic 在 API 与企业预算上直接竞争。
  • 消费应用层：豆包、Kimi 与 ChatGPT、Claude、Gemini App、Perplexity 在用户时长与留存上竞争。
  • Coding 层：Cursor 与 GitHub Copilot 直接竞争开发者席位与团队预算。
  • 视角按钮：投资视角更强调上游与模型层，企业采购视角强调可落地与可集成能力。

OpenClaw 在产业中的定位

按常见语义，OpenClaw 更像开源 Agent/工具编排框架，位置通常在“应用中间层（Agent Runtime/Workflow）”，上游依赖基座模型与云算力，下游服务企业自动化场景。如果你指的是另一个同名项目，可按具体产品功能再精确归位。
